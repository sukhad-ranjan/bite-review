# Bite Review 🍽️

Bite Review is a web application that allows students to rate and review food from different canteens on campus.

---

## 🚀 Features

- ⭐ Rate food items
- 💬 Submit feedback/comments
- 📊 View average ratings
- 🔥 Discover trending dishes
- 🔐 Admin login for canteen owners
- 📈 Weekly reports & analytics
- 📄 Export reports as PDF
- 📧 Email report functionality

---

## 🛠️ Tech Stack

- React (Vite)
- CSS

### Backend
- PHP

## ⚙️ Setup (Local Development)

### 1. Clone the repository
git clone https://github.com/sukhad-ranjan/bite-review.git
cd bite-review

### 2. Install frontend dependencies
npm install

### 3. Run frontend
npm run dev

### 4. Run backend server
cd api
php -S localhost:8000
